A simple, lightweight, and nostalgic **Snake Game** built using Python's built-in `turtle` library. Navigate the snake, eat the food, and grow longer without hitting the walls or yourself!

---

## 🎮 How to Play
- ** W **: Move Up
- ** S **: Move Down
- ** A **: Move Left
- ** D **: Move Right

## 🚀 Features
- **Classic Gameplay:** Faithful recreation of the classic arcade experience.
- **Dynamic Scoring:** Keep track of your score as you eat food.
- **Collision Detection:** Game Over logic for hitting walls or the snake's own tail.
- **Turtle Power:** Built entirely with Python's `turtle` module (no external dependencies required).

---

## 🛠 Prerequisites
You only need Python installed on your system. No external libraries are required!
